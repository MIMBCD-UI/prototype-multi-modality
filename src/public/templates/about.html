About


This version is based on the Cornerstone open source project. All 2D image rendering is done on the client side side using HTML5 and JavaScript. DICOM P10 Instances are retrieved via WADO, parsed and displayed by the viewer - the server side performs no processing.

The demo is developed under the MIMBCD-UI Project. The Project is hosted by ISR-Lisboa in collaboration with M-ITI and INESC-ID. Both ISR-Lisboa and M-ITI are interdisciplinary research laboratories from LARSyS. ISR-Lisboa and INESC-ID are associated laboratories of IST from ULisboa. The research is part of a protocol between ISR-Lisboa and Hospital Fernando Fonseca (HFF).
